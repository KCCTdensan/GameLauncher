弾幕YOKE Ver 1.0.12

Readme.txt はじめにお読みください
----------------------------------------------------------------------------------------------------

・概要
	弾幕から避け続け，無限に耐久をしてみたり旗を回収するゲームです。
	自分から攻撃をすることはありません。弾幕を楽しみたい人向けです。
	難易度選択で真ん中に行くほど簡単になり，端を選ぶほど難しくなります。
	難易度ごとに，得点の倍率も変わるのでベストスコアを目指してください！

・導入方法
	[弾幕yoke.exe]があるディレクトリをお好みの場所においてください。

・アンインストール方法
	[弾幕yoke.exe]があるディレクトリを削除するだけです。

・操作方法
	・移動		[方向パッド]
	・途中離脱	[ESC]
	・続行		[ENTER]

・バグ・報告
	現在は報告の手段は実装されていません。
